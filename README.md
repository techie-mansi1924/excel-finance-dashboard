# 📊 Interactive Finance Dashboard in Microsoft Excel

An interactive Finance Dashboard built in Microsoft Excel to analyze Revenue, Expenses, and Profit using Pivot Tables, Pivot Charts, KPIs, and Slicers.

This project demonstrates Excel dashboard development and business reporting skills commonly required in Data Analyst roles.

# Project Overview

The dashboard provides a complete overview of business financial performance with dynamic filtering options.

Users can analyze:

- Revenue
- Expenses
- Profit & Loss
- Department-wise Performance
- Monthly Profit Trend
- Revenue Category Distribution
- Expense Category Distribution

using interactive slicers.

# Dataset Information

The dataset contains business transactions with the following fields:

- Transaction ID
- Date
- Month
- Quarter
- Year
- Department
- Transaction Type
- Category
- Amount
- Payment Method
- Approval Status
- Responsible Person

The dashboard updates automatically using Pivot Tables and Slicers.
